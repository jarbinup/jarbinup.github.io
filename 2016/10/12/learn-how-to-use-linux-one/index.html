基本概念

  1. Linux 和 Windows 一样是一种操作系统。在计算组成中对应内核和系统调用两层。其上是应用程序层，其下是硬件层。
  2. Linux 本身只是操作系统的内核。内核是使其他程序能够运行的基础。它实现了多任务和硬件管理。
  3. 命令行解释器（shell），可以理解为是包含在内核之上的一层，是指“提供给使用者使用界面”的软件（命令解析器），用户通过 shell 和内核进行交互，不会直接去调用内核程序。
  4. 在 UNIX/Linux 中比较流行的常见的 Shell 有 bash，zsh，ksh，csh 等，ubuntu 终端默认使用的是 bash.
  5. linux 本身是没有图形化界面的，在发行版上看到的界面是运行在其上的一套软件。和 window 桌面之间的区别如下图：
  6. 终端本质上是对应着 Linux 上的 /dev/tty 设备, Linux 默认提供了 6 个纯命令行界面的 “terminal”（准确的说这里应该是 6 个 virtual consoles）来让用户登录。可以通过Ctrl+Alt+F1～F6 进行切换。Ctrl+Alt+F7 对应图形化界面。图形化界面中我们所说的终端其实叫”终端模拟器”，比如 ubuntu 下的 gnome-terminal

用户管理

查看用户

  1. who am i 返回当前伪终端用户的用户名
  2. whoami 返回当前登录用户的用户名

创建用户

创建用户需要 root 权限，而一般我们是以普通用户(或者管理员)登录系统的。所以这里要使用 sudo 命令了。
sudo 命令的使用前提：a.当前用户的登录密码， b. 当前用户属于 sudo 用户组。

快捷键 功能
su 切换到 user 用户
su - 同时切换环境变量为目标用户的环境变量
sudo 以特权级别运行 cmd
