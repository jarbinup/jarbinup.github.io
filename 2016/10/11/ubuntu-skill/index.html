ubuntn 实用技巧

发表于 2016-10-11   |   分类于 工具   |  

总结积累常用的 ubuntn 技巧( 持续更新中 )：

桌面技巧

快捷键 功能
ctrl+alt+T 直接打开终端
alt+F4 关闭当前窗口
super(保持) 打开启动器
super+tab 启动器中切换
super+1到9 直接打开
super+T 打开垃圾桶
super(轻触) 打开 dash 面板
ctrl+tab 切换
alt+tab 切换应用程序
alt+ 切换当前应用程序窗口
super+w 展示所有窗口

终端技巧

快捷键 功能
cd { dirname } 进入某个文件夹
tab 自动补全
cd .. 返回上一层目录
xdg-open { file 或 URL } 用命令行打开相应的文件
ctrl+shift+N 在当前路径下新建一个终端，注意和 ctrl+alt+T 区别
ctrl+D 表示键盘输入或退出终端
exit 表示键盘输入或退出终端 ,和 alt+F4 不同
Ctrl+s 暂定当前程序，暂停后按下任意键恢复运行
Ctrl+z 将当前程序放到后台运行，恢复到前台为命令fg
Ctrl+a 将光标移至输入行头，相当于Home键
Ctrl+e 将光标移至输入行末，相当于End键
Ctrl+k 删除从光标所在位置到行末
Alt+Backspace 向前删除一个单词
Shift+PgUp 将终端显示向上滚动
Shift+PgDn 将终端显示向下滚动
man Manual Page 获得命令的使用方式和说明

Shell 常用通配符：

字符 含义
* 匹配 0 或多个字符
? 匹配任意一个字符
[list] 匹配 list 中的任意单一字符
[!list] 匹配 除list 中的任意单一字符以外的字符
[c1-c2] 匹配 c1-c2 中的任意单一字符 如：[0-9] [a-z]
{string1,string2,…} 匹配 sring1 或 string2 (或更多)其一字符串
{c1..c2} 匹配 c1-c2 中全部字符 如{1..10}
